# Contribuindo

## Início Rápido

```bash
git clone https://github.com/AndreBFarias/Detector-de-Doppelganger.git
cd Detector-de-Doppelganger
./install.sh
```

## Como Contribuir

1. Faça um fork do projeto.
2. Crie uma branch para sua feature (`git checkout -b feature/nome`).
3. Implemente e teste localmente.
4. Commit com mensagem descritiva (`git commit -m 'feat: descrição'`).
5. Push e abra um Pull Request.

## Checklist

- [ ] Fork e branch criados
- [ ] Testes passando
- [ ] Sem regressões na pipeline de inferência
- [ ] PR com descrição do que foi alterado
